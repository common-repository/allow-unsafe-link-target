Prevent WordPress from adding rel="noopener noreferrer" to links with target="_blank".

== Description ==

Removes rel="noopener noreferrer" from links opened in a new window / tab.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/allow-unsafe-link-target` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Done.
